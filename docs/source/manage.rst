Description
-----------

`manage.py` is the command-line utility that lets you interact with this Django project in various ways. It provides several commands for managing the project, including running the development server, applying migrations, and more.

Usage
-----

You can use the following command to run the development server:

.. code-block:: shell

   python manage.py runserver

Other commonly used commands include:

- `migrate`: Applies database migrations.
- `createsuperuser`: Creates a superuser account for the Django admin.
- `startapp`: Creates a new Django app within the project.
